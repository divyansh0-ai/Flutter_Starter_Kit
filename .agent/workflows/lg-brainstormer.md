Now that the project is initialized, let's think about the "Wow-Factor".

## Goals

## Checklist

- [ ] Map out the UI flow.
- [ ] Identify which Gemini model to use (e.g., `gemini-1.5-flash`).

> [!IMPORTANT]
> A Liquid Galaxy app is about immersion. Think BIG (literally, across all screens).
